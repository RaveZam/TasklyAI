How It Works

Users log in using Supabase Auth.

Each user gets their own workspace.

Tasks are stored in Supabase and displayed on a drag-and-drop kanban board.

The AI button helps generate tasks or break down ideas using natural language.

Target Users

Students managing schoolwork

Small teams avoiding complex tools

Solo developers tracking personal projects

Simple. Fast. No clutter.
